# 需求文档：IELTS Speaking Coach v0.1

## 一、项目目标

做一个雅思口语文本练习系统，帮助用户选择题目、提交回答、获得评分建议，并保存练习记录。

v0.1 的目标不是做完整产品，而是训练企业后端开发的基本闭环：

```text
接口设计 → 业务逻辑 → 数据库持久化 → 异常处理 → 文档说明
```

## 二、用户故事

### 1. 查询题库

作为用户，我希望能看到系统内置的雅思口语题目，以便选择一道题进行练习。

### 2. 提交回答

作为用户，我希望提交一段英文回答，并获得系统反馈。

### 3. 查看历史记录

作为用户，我希望查看自己之前提交过的练习记录。

## 三、当前版本范围

包含：

- 题库查询
- 文本回答提交
- 模拟评分
- 练习记录保存
- 历史记录查询

不包含：

- 用户登录
- 真实大模型评分
- 语音上传
- ASR 转写
- Agent 编排
- 前端页面

这些会放到后续版本。
