# 数据库设计：IELTS Speaking Coach v0.1

## 1. questions：题库表

| 字段 | 类型 | 说明 |
|---|---|---|
| id | Integer | 主键 |
| part | String | 雅思口语 Part 1 / Part 2 / Part 3 |
| topic | String | 题目主题 |
| content | Text | 题目内容 |

## 2. practice_records：练习记录表

| 字段 | 类型 | 说明 |
|---|---|---|
| id | Integer | 主键 |
| question_id | Integer | 题目 ID |
| answer | Text | 用户回答 |
| score | String | 模拟评分 |
| feedback | Text | 模拟反馈 |
| created_at | DateTime | 创建时间 |

## 3. 当前设计说明

v0.1 暂时没有用户系统，所以 practice_records 没有 user_id。

后续 v0.4 增加登录后，需要加入：

```text
users 表
practice_records.user_id 字段
```
