 # 模型接入说明

当前后端使用 OpenAI-compatible Chat Completions 接口。

## 1. DeepSeek 本地试跑

复制 `.env.example` 为 `.env`，然后填写：

```text
LLM_PROVIDER=deepseek
DEEPSEEK_API_KEY=你的 DeepSeek API Key
DEEPSEEK_BASE_URL=https://api.deepseek.com
DEEPSEEK_MODEL=deepseek-v4-flash
```

重启后端：

```powershell
.\.venv\Scripts\python.exe -m uvicorn app.main:app --reload --port 8000
```

## 2. 切换模型

如果要换成其他 OpenAI-compatible 服务，可以使用通用配置：

```text
LLM_PROVIDER=custom
LLM_API_KEY=你的 API Key
LLM_BASE_URL=https://example.com/v1
LLM_MODEL=your-model-name
```

## 3. Fallback 行为

如果没有配置 API Key，或模型调用失败，接口不会崩溃。

- 练习追问会使用本地模板 fallback
- 练习结束评价会使用本地规则 fallback

后端终端会打印简短日志，方便判断是否走到了真实模型。

## 4. 注意事项

- 不要把真实 `.env` 提交到 Git 或发给别人。
- API Key 只放后端，不能放前端。
- 修改 `.env` 后需要重启后端服务。
