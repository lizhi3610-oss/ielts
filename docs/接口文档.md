# 接口文档：IELTS Speaking Coach v0.1

## 1. 健康检查

```text
GET /health
```

响应示例：

```json
{
  "status": "ok",
  "app": "IELTS Speaking Coach",
  "version": "0.1.0"
}
```

## 2. 查询题库

```text
GET /questions
```

响应示例：

```json
[
  {
    "id": 1,
    "part": "Part 1",
    "topic": "Work or Study",
    "content": "Do you work or are you a student?"
  }
]
```

## 3. 提交练习回答

```text
POST /practice/submit
```

请求体：

```json
{
  "question_id": 1,
  "answer": "I am currently working as a software engineer..."
}
```

响应示例：

```json
{
  "question_id": 1,
  "answer": "I am currently working as a software engineer...",
  "score": "6.0",
  "feedback": "你的回答基本完整，但表达还可以更自然。建议增加连接词，并使用更丰富的词汇。"
}
```

异常情况：

| 场景 | HTTP 状态码 | 说明 |
|---|---:|---|
| 题目不存在 | 400 | question_id 不存在 |
| 回答为空 | 422 / 400 | Pydantic 或业务校验失败 |

## 4. 查询练习记录

```text
GET /practice/records
```

响应示例：

```json
[
  {
    "id": 1,
    "question_id": 1,
    "answer": "I am currently working as a software engineer...",
    "score": "6.0",
    "feedback": "你的回答基本完整...",
    "created_at": "2026-05-20T12:00:00"
  }
]
```
